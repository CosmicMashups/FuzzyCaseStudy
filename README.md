# Fuzzy Logic Computation Simulator

Presentation-style interactive web app that simulates the fuzzy logic computation process step by step (crisp input, fuzzification, rule evaluation, aggregation, defuzzification, final output). For educational use in classroom or thesis defense.

## Run

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (e.g. http://localhost:5173).

## Build

```bash
npm run build
```

## Stack

- React 18, TypeScript, Vite, TailwindCSS, Framer Motion, Recharts, Zustand. No backend.

## Test cases

Presets A–D match the reference notebook outputs (Monitor & Self-Care, Schedule Counseling, Urgent Referral, Emergency Protocol).
